# Bubble Sort
* best case: O(n)
* worst case: O(n^2)

# Selection Sort
* O(n^2)

# Insertion Sort
* O(n^2)
* but usually the fastest than Bubble sort, Selection sort

# Merge Sort
* O(nlog n)

# Quick Sort
* usually: O(nlog n)
* worst case: O(n^2) - when the array is already sorted

# Radix Sort
* O(d(n + r)) - r = 2, common
